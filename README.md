# Pepsi 首席铲屎官指南
---
## 投喂指南
1. 干粮：一天一猫碗，大约65g/天，可以分2-4次投喂
2. 罐罐：一天最多一个罐罐，如果投喂了罐罐，可以适当减少一半干粮，一周可以喂0-4次罐罐
3. 水：每天换两次水，用过滤水
4. 零食：0-10小块一天 可以不投喂

## 铲屎
可以1-4天铲屎一次，根据铲屎官的时间来安排
